# The BigCo PM Field Guide

### 📖 **[Read the guide →](https://umahkhan.github.io/product-management-101/)**

A single-page reference on product management in large technology companies. Free to read, no sign-up.

> **Note:** clicking `index.html` above shows you the source code — GitHub doesn't render HTML files. Use the link above to read it. To read it offline, download `index.html` and open it in any browser: no build step, no dependencies, no network requests.

## What it is

A working reference organised as a learning progression rather than a slide dump. Twenty-seven sections across six parts:

| Part | Covers |
|---|---|
| **I · Foundations** | Value creation and capture (WTP/WTS), the PM's three curves, economies of scale and minimum efficient scale, network effects, platform vs. linear business models |
| **II · The BigCo** | Why large firms diversify, the three NPV conditions for value-adding scope, the winner's curse, four synergies and five diseconomies, build/buy/partner, organisation structure, the five levers |
| **III · Evolution** | S-curves and the technology life cycle, dominant design, the innovator's dilemma, the adoption curve and Moore's chasm, the product performance curve |
| **IV · The Value Cycle** | The Product Value Cycle, the twelve questions, FAEEP feature analysis, customer and profit value propositions, protecting value from imitation, hold-up, and substitution |
| **V · Cases** | Netflix end-to-end, the Metaverse as anatomy of a failure, and a twelve-entry failure gallery indexed by which question each product failed |
| **VI · Toolkit** | Decision frameworks, the product lifecycle as responsibilities, glossary |

If you have one hour, read §01, §02, §18, and §19.

## Features

- **Progressive disclosure** — overview prose with "go deeper" sections for the detail
- **17 original diagrams** — S-curves, the adoption curve, the value stack, network-effect scoring, org structures, the Product Value Cycle, and an interactive minimum-efficient-scale chart
- **Light and dark themes**, following your system preference with a manual toggle
- **Accessible** — keyboard-navigable charts, labelled figures, colourblind-safe palette validated for CVD separation and contrast in both themes
- **Self-contained** — one HTML file, ~200 KB, works offline

## Attribution

Synthesised from the lecture materials of **TECH 5920: BigCo Studio**, taught by **Professor Gautam Ahuja** at Cornell University. The frameworks, case selections, and much of the analytical sequencing are his.

The underlying frameworks draw on Abernathy & Utterback, Christensen, Foster, Dosi, Moore, Rogers, Porter, Kotler, and Ahuja & Novelli (2017), *Redirecting Research Efforts on the Diversification–Performance Linkage*.

This is a personal study reference. It reorganises and paraphrases course material rather than reproducing it; practice examination questions and duplicated review slides are excluded. It is **not** an official course artifact, is not endorsed by the instructor or by Cornell, and is not licensed for redistribution.
